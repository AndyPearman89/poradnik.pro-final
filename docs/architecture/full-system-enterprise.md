# Poradnik.pro - Full System Enterprise (AdSense + Afiliacja + Lead Engine)

## 1. Core model

Poradnik.pro jest systemem maksymalizacji wartosci uzytkownika.

FLOW:

User -> Content -> Decision -> Monetization

Sciezki:

- AdSense (niska intencja)
- Afiliacja (srednia intencja)
- Lead (wysoka intencja)

## 2. Intent segmentation engine

LOW INTENT:

- edukacja
- ciekawosc
- monetyzacja: AdSense

MID INTENT:

- wybor produktu
- monetyzacja: afiliacja

HIGH INTENT:

- potrzeba uslugi
- monetyzacja: lead

## 3. AdSense system

Cel: skalowalny cashflow.

Placement:

- po 2 akapicie
- po kazdym H2
- koniec artykulu

Struktura:

intro
[ADS]
sekcja
[ADS]
sekcja
[ADS]

Mobile:

- in-content ads
- opcjonalnie sticky ads

KPI:

- RPM: 10-40 PLN
- CTR: 1-3%

Zasada: ads nie moga zabijac UX ani flow leadowego.

## 4. Afiliacja system

Cel: monetyzacja decyzji zakupowej.

Typy contentu:

- rankingi
- porownania
- tresci typu co kupic

Elementy:

- TOP 3
- tabela porownawcza
- box rekomendowany

CTA:

- Sprawdz cene
- Zobacz oferte

KPI:

- CR: 2-10%
- EPC

## 5. Lead engine system

Cel: maksymalny przychod.

Flow:

User -> formularz -> routing -> firmy -> odpowiedzi

Formularz:

- problem
- lokalizacja
- kontakt

Routing:

- 3-5 firm
- opcja exclusive

CTA:

- Znajdz specjaliste
- Porownaj oferty

KPI:

- CR: 3-15%
- wartosc leada: 10-200+ PLN

## 6. Page architecture

PORADNIK:

- 50% content
- 25% ads
- 25% afiliacja + CTA

RANKING:

- 60% afiliacja
- 30% lead
- 10% ads

LOKALNA:

- 80% lead
- 20% ads

## 7. SEO engine

Typy stron:

- poradniki
- rankingi
- porownania
- lokalne

Generator:

- problem + rozwiazanie
- usluga + miasto
- ranking + miasto

Skala:

100 -> 1k -> 10k -> 100k+

## 8. UX i conversion

Zasady:

- CTA co 2 sekcje
- szybkie decyzje
- trust: opinie i oceny

Anty-wzorce:

- brak CTA
- przeladowanie reklamami

## 9. Automation system

- AI generuje tresci
- auto ads placement
- auto afiliacja
- auto CTA

## 10. Traffic channels

- SEO (core)
- YouTube Shorts
- Pinterest
- Google Discover

## 11. Metrics

- RPM
- CTR

AFILIACJA:

- EPC

LEAD:

- CR

GLOBAL:
